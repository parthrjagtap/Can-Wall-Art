# Can Wall Art
Wanted to make can wall in my room, this helps create a mockup.

How it works?

1. Select the soda you want.
2. Click on the cell / can position you want to fill.
3. Paint what you want.

Additional features:
1. Fill All  - Fills all the cells with the selected can.
2. Clear All - Clears all the cells.
3. Presets   - a. Diagonal: Fills the diagonally with the sodas
               b. DE flag:  Creates the Deutschland(German) flag with cans closer to the colours
               c. Random:   Creates a wall with random sodas
